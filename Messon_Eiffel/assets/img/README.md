# Carpeta de imágenes

Aquí van las fotos reales del negocio cuando estén disponibles.

## Imágenes esperadas

| Archivo | Uso |
|---------|-----|
| `hero.jpg` | Foto principal del hero (portada) — recomendado: 1800×1200px |
| `reservas-bg.jpg` | Fondo de la sección de reservas — recomendado: 1600×900px |
| `logo.png` | Logo del mesón en alta resolución |
| `local-1.jpg` | Foto del interior del local |
| `platos/` | Subcarpeta con fotos de platos (opcional) |

## Notas

- Formatos recomendados: `.jpg` para fotos, `.png` para logo/transparencias, `.webp` para mejor rendimiento
- Tamaño máximo recomendado por imagen: 500 KB
- Mientras no haya fotos, la web usa imágenes de Unsplash como placeholder
